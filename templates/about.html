Marvel-Authentication was a Graphical Password Authentication platform designed to overcome difficulties raised from the normal
Text - based Password Authentication system. Register an  account by  entering  the Username,  email  and select  the grids for the
Password. Then login to See  the change in  Marvel-Auth status. Remember  the email registered with the password to  receive the
Notification  mail on reset password page. Marvel-Auth was a  Marvel  theme based  Graphical Authentication System and it can be
of anytype. HTML, CSS and Django was used to develop this  Authentication System. Modifications to the Marvel-Auth was allowed❣
